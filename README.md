## Yeni Next.js Projesi

Bu repo, sıfırdan proje geliştirmek için sadeleştirilmiş bir Next.js başlangıç iskeletidir.

### Çalıştırma

Geliştirme sunucusunu başlatmak için:

```bash
npm install
npm run dev
```

Ardından tarayıcınızda `http://localhost:3000` adresini açabilirsiniz.

### Proje Yapısı

- `src/app/layout.js`: Uygulama genel iskeleti ve metadata tanımı.
- `src/app/page.js`: Varsayılan ana sayfa bileşeni. Kendi sayfalarınız ve bileşenleriniz için buradan başlayabilirsiniz.
- `src/app/globals.css`: Global stiller.

İhtiyaç duymadığınız dosyaları özgürce kaldırıp yapıyı projenize göre özelleştirebilirsiniz.
